# Git Operations Skill

## Purpose

Perform common Git operations for repository management.

## Procedure

### Step 1: Pre-flight Checks

Verify the working tree is clean and the branch is up to date.

### Step 2: Merge Conflict Resolution

Identify conflicting files using `git status`, open each in the editor, resolve markers,
and stage resolved files. This step covers a distinct topic (conflict resolution) not
previously established by the parent context — the heading is not redundant.

### Step 3: Post-merge Verification

Run tests and confirm the merge result is correct.
