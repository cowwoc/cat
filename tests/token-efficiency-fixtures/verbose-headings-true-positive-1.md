# Configuration Analysis Skill

## Purpose

Analyze configuration files and report validation findings.

## Procedure

### Step 1: Configuration Analysis Steps

Read all configuration files in the project root and validate each one.

### Step 2: Configuration Analysis Results Reporting

After reading the files, report which ones passed and which failed validation.
