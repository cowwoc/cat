# Dependency Audit Skill

## Purpose

Audit project dependencies for security vulnerabilities and outdated versions.

## Procedure

### Step 1: Dependency Audit Scanning Phase Steps

Scan all dependency manifests (package.json, pom.xml, requirements.txt).

### Step 2: Dependency Audit Vulnerability Detection Steps

Cross-reference scanned dependencies against known CVE databases.

### Step 3: Dependency Audit Report Generation Steps

Produce a ranked list of vulnerabilities with remediation guidance.
