# Skill Invocation

You have access to skills via the Skill tool. Before performing any action, check if a skill below matches what you are
about to do. If it matches, this is a BLOCKING REQUIREMENT: invoke the Skill tool BEFORE performing that action.

**How to invoke:** Use the Skill tool with the skill name (e.g., `skill: "cat:git-commit"`).

NEVER perform an action that a skill covers without invoking the skill first. NEVER mention a skill without actually
calling the Skill tool.
