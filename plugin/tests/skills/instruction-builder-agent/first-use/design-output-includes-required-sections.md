## Turn 1

Design a skill for a CAT plugin agent that creates a new git branch and commits a file. The skill must
follow the standard CAT plugin skill structure with sequential step numbering starting at Step 1.

## Assertions

1. The Skill tool was invoked
2. The skill draft includes a Purpose section
3. The skill draft includes a Procedure section
4. The skill draft includes a Verification section
5. The first step in the Procedure section is numbered Step 1
6. The skill draft does not contain a step numbered Step 0
